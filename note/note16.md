## 前置知识

## 矩阵与向量偏导

### 一、核心约定
全程使用工程/机器学习最常用的「分母布局」（无需掌握其他布局，避免混乱）：
- 向量 x ∈ Rⁿ：默认是「列向量」（书写为 x = [x₁, x₂, ..., xₙ]ᵀ）
- 标量对向量求导：结果是「列向量」（和自变量向量形状一致）
- 向量对向量求导：结果是「矩阵」（行数=因变量向量维度，列数=自变量向量维度）
- 补充：本文所有求导均基于「偏导」，符号 ∂/∂x 均表示对向量 x 中每个分量求偏导后，按规则拼接结果

---

### 二、核心求导类型

#### 类型1：标量对向量求导（最常用，结果为向量）
定义：标量 f（单个数字），列向量 x = [x₁, x₂, ..., xₙ]ᵀ，求 ∂f/∂x，本质是「对每个 xᵢ 求偏导，再拼成列向量」。

##### 1.1 线性形式：f = aᵀx（a 是与 x 同维度的列向量）
手推步骤：
1. 展开式子：aᵀ = [a₁, a₂, ..., aₙ]，因此 f = a₁x₁ + a₂x₂ + ... + aₙxₙ
2. 对每个 xᵢ 求偏导：∂f/∂xᵢ = aᵢ（因为只有 aᵢxᵢ 含 xᵢ，其余项导数为0）
3. 拼接结果：将所有偏导拼成列向量，即 ∂(aᵀx)/∂x = [a₁, a₂, ..., aₙ]ᵀ = a

实例（带具体数字）：
设 a = [3, 4]ᵀ，x = [x₁, x₂]ᵀ，则 f = aᵀx = 3x₁ + 4x₂
求导：∂f/∂x₁ = 3，∂f/∂x₂ = 4，因此 ∂f/∂x = [3, 4]ᵀ = a

##### 1.2 二次型形式：f = xᵀAx（A 是 n×n 方阵）
手推步骤：
1. 展开式子：f = Σᵢⱼ Aᵢⱼ xᵢ xⱼ（Aᵢⱼ 是矩阵 A 第 i 行第 j 列的元素）
2. 对某个 xₖ 求偏导：∂f/∂xₖ = Σⱼ Aₖⱼxⱼ + Σᵢ Aᵢₖxᵢ（拆分含 xₖ 的项，链式法则求导）
3. 拼接结果：整理后得 ∂(xᵀAx)/∂x = (A + Aᵀ)x（Aᵀ 是 A 的转置）
4. 特殊情况（常用）：若 A 是对称矩阵（Aᵀ = A，机器学习中几乎都是），则 ∂(xᵀAx)/∂x = 2Ax

实例（带具体数字）：
设 A = [[1, 0], [0, 1]]（对称矩阵），x = [x₁, x₂]ᵀ，则 f = xᵀAx = x₁² + x₂²
求导：∂f/∂x₁ = 2x₁，∂f/∂x₂ = 2x₂，因此 ∂f/∂x = [2x₁, 2x₂]ᵀ = 2Ax

##### 1.3 高频组合：f = ||Ax - b||²（范数平方，机器学习损失函数核心）
先明确：||向量||² 表示「向量中每个元素平方后相加」（2-范数的平方）

手推步骤：
1. 展开式子：设 A 是 m×n 矩阵，x ∈ Rⁿ，b ∈ Rᵐ，则 Ax - b = [A₁₁x₁ + ... + A₁ₙxₙ - b₁, ..., Aₘ₁x₁ + ... + Aₘₙxₙ - bₘ]ᵀ
2. 范数平方展开：f = ||Ax - b||² = Σₖ (Aₖ₁x₁ + ... + Aₖₙxₙ - bₖ)²
3. 对每个 xᵢ 求偏导（链式法则）：∂f/∂xᵢ = 2 Σₖ (Aₖ₁x₁ + ... + Aₖₙxₙ - bₖ) · Aₖᵢ
4. 整理成矩阵形式：∂||Ax - b||²/∂x = 2Aᵀ(Ax - b)

实例（带具体数字）：
设 A = [[1, 2], [3, 4]]，x = [x₁, x₂]ᵀ，b = [5, 6]ᵀ
1. 计算 Ax - b = [x₁ + 2x₂ - 5, 3x₁ + 4x₂ - 6]ᵀ
2. 范数平方展开：f = (x₁ + 2x₂ - 5)² + (3x₁ + 4x₂ - 6)²
3. 求偏导：
   ∂f/∂x₁ = 2(x₁ + 2x₂ - 5)·1 + 2(3x₁ + 4x₂ - 6)·3 = 20x₁ + 28x₂ - 46
   ∂f/∂x₂ = 2(x₁ + 2x₂ - 5)·2 + 2(3x₁ + 4x₂ - 6)·4 = 28x₁ + 40x₂ - 68
4. 矩阵形式验证：2Aᵀ(Ax - b) = 2[[1, 3], [2, 4]]·[x₁ + 2x₂ - 5, 3x₁ + 4x₂ - 6]ᵀ = [20x₁ + 28x₂ - 46, 28x₁ + 40x₂ - 68]ᵀ，与偏导结果一致

#### 类型2：向量对向量求导（结果为矩阵）
定义：列向量 y ∈ Rᵐ（因变量，y = [y₁, y₂, ..., yₘ]ᵀ），列向量 x ∈ Rⁿ（自变量），求 ∂y/∂x，本质是「每个 yᵢ 对每个 xⱼ 求偏导，拼成 m×n 矩阵」。

核心形式：y = Ax（A 是 m×n 矩阵）

手推步骤：
1. 展开式子：y = Ax 可拆分为 yᵢ = Σⱼ Aᵢⱼxⱼ（每个 yᵢ 是 x 分量的线性组合）
2. 求偏导：对每个 yᵢ 求每个 xⱼ 的偏导，得 ∂yᵢ/∂xⱼ = Aᵢⱼ（线性项求导，系数即为导数）
3. 拼接结果：将所有 ∂yᵢ/∂xⱼ 拼成矩阵，行对应 yᵢ，列对应 xⱼ，即 ∂(Ax)/∂x = A

实例（带具体数字）：
设 A = [[1, 2], [3, 4]]，x = [x₁, x₂]ᵀ，则 y = Ax = [y₁, y₂]ᵀ = [x₁ + 2x₂, 3x₁ + 4x₂]ᵀ
1. 求所有偏导：
   ∂y₁/∂x₁ = 1，∂y₁/∂x₂ = 2
   ∂y₂/∂x₁ = 3，∂y₂/∂x₂ = 4
2. 拼接成矩阵：∂y/∂x = [[1, 2], [3, 4]] = A

---

### 三、核心公式总结
（均基于分母布局，列向量约定）
1. 标量对向量（线性）：∂(aᵀx)/∂x = a（a 为列向量）
2. 标量对向量（二次型）：∂(xᵀAx)/∂x = (A + Aᵀ)x；若 A 对称，∂(xᵀAx)/∂x = 2Ax
3. 标量对向量（范数平方，高频）：∂||Ax - b||²/∂x = 2Aᵀ(Ax - b)
4. 向量对向量（线性）：∂(Ax)/∂x = A（A 为因变量维度 × 自变量维度矩阵）

---

### 四、关键易错点
- 约定不能乱：本文分母布局是工程/ML通用，若换成分子布局，结果形状会变（比如标量对向量求导会变成行向量），无需掌握其他布局，记死本文约定即可。
- 向量默认是列向量：书写时 x 是列向量，xᵀ 才是行向量，展开式子时不要搞反行列。
- 范数平方的本质：||向量||² 就是“元素平方和”，不是向量平方（向量不能直接平方），展开后就是普通的多元函数，求导用高中链式法则即可。
- 矩阵转置的作用：二次型、范数平方求导后会出现 Aᵀ，核心是“调整矩阵维度”，确保结果形状符合约定（比如标量对向量求导结果是列向量）。
