##**Reasons For Joining QG-AI**

*我想我在决定参加寒假训练营之前确实think了很多了，现在正是do的时候了。*:boxing_glove:

想要加入QG的人工智能组并非一朝一夕的想法，刚开始了解到QG是在25年国庆那会，当时觉得很酷吧，这么厉害的团队我竟然有机会成为其中的一员。但随着日子一天一天过去，这份冲动冷却了——我不想做出一个随意的决定。有时半夜依旧清醒时，脑海中各种想法开始相互碰撞，翻来覆去，然后，我开始意识到“为什么‘’很重要，是的，我得给自己找一个想加入QG充分且足够坚定的动机。

*总有些原则是更高的，正如我认为”行动是为理想服务的“。*:clinking_glasses:

网络时代的发展太快了，快到让人容易迷失方向，迷失自我。12月份的职规作业恰是一次准时的转机，我知道自己真正的爱好并不多，赛车，准确来说，世界一级方程式赛车，是如此让我着迷。但在职规这档事之前，我从未将其与自己的梦想相联系：我想成为一名F1比赛策略师。

我开始更深入了解这相关的一切，也逐渐意识到加入QG-AI将有助于我实现自己的梦想，而这正是那个动机。这个梦想对于我而言就像是乌托邦吧，但我想在梦想消散之前，真正的，去试一次。
